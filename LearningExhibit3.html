Regex and the Republic of Texas Dataset

This was a very challenging exercise for someone like myself who had never even heard of Regex before. My initial notes on this exercise can be found here. The difficulties I had with this exercise started rather early on
but in the end I took my time, powered through, and learned a lot in the process.

My troubles started at the end of step one. RegExr became rather frustrating for me as many of the commands did not match the ones I was using in Notepad++. While I understood how this program could be useful for
testing your strings before executing them within your document and potentially mucking it all up, overall, since the more complex commands did not match the ones I was using, I quickly gave up on RegExr.
Furthermore, when using Notpead++ you can test your string one line at a time before committing to applying it to the entire document. Also, Notepad++ also features a handy little "undo" button in the top
menu bar.




The next problem I had, and the one that I could not find a solution to without getting creative on my own and finally ignoring the instructions provided, was removing the blank lines in step 3. The
code given in the instructions simply did not work no matter how many variations of it I tried (\n\r, ^\r, and any variations of this). The tutorial video did not help either. What was beig demonstrated there simply did not
work for me. Googling it did not help either, as Google results suggested using the same code as was suggested in the textbook exercise. Finally I figured out on my own how to accomplish this without using any code at
all, which may have something to do with the reason I could not get it to work using code in the first place.




The next big snag I hit was in step 4. And that was not just part of step 4, but all of it. In the end, I had to watch the tutorial video to finally get it all right. but this was after being rather
stubborn and wasting a lot of time with trial and error.


Conclusion

This was a very challenging exercise for me. But, I knew that this was going to be very useful for me in my final project, and so, stuck with it in spite of much frustration. In the end, it all worked out,
